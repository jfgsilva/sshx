# sshx

Cli for managing ssh connections

## Usage

### Add server

```bash
sshx add
```

### Connect to server

```bash
sshx -name {server_name} connect
```

or just

```bash
sshx
```

and select server from the list

### List added servers

```bash
sshx list
```

### Remove servers

```bash
sshx rm
```

### Update server info

```bash
sshx update
```
